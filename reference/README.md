# Reference Directory

This directory contains the public diligence summaries for the repository.

These files are meant to be lighter than raw code, but stricter than marketing copy.

## What Lives Here

- `ASSURANCE_BOARD.md`
  - current public summary of tests, formal coverage, chaos coverage, and live evidence posture
- `CHAOS_MATRIX.md`
  - current public summary of the deterministic local chaos experiments

## How To Use It

- start here after reading the root `README.md`
- use `ASSURANCE_BOARD.md` for a fast technical diligence snapshot
- use `CHAOS_MATRIX.md` to understand how failure cases are being rehearsed

## Boundary

This directory is public and tracked.

It is intentionally separate from local-only diligence drafts, working notes, and run artifacts that are kept out of the Git tree.
